# **App Name**: GeoCatastro España

## Core Features:

- Input de Referencia Catastral: Permite al usuario ingresar una referencia catastral para buscar propiedades en España.
- Búsqueda de Datos Catastrales: Busca la dirección, latitud, longitud y altitud asociadas a la referencia catastral proporcionada a través de la API externa del Catastro de España.
- Visualización de Detalles de la Propiedad: Muestra los datos recuperados (dirección, latitud, longitud en decimales y altitud) de forma clara y estructurada en la interfaz de usuario.
- Herramienta de Análisis Geográfico Asistido por IA: Utiliza un tool de IA para generar una breve descripción o información contextual sobre la ubicación geográfica de la propiedad, basándose en la latitud y longitud obtenidas.

## Style Guidelines:

- Esquema de color general: Se utiliza un esquema de color claro para fomentar la legibilidad y la profesionalidad. Los colores están inspirados en la confiabilidad de la información oficial y los elementos geográficos.
- Color principal: Azul profesional (#2952A3), simbolizando fiabilidad y estabilidad, utilizado para elementos clave de la interfaz como botones y encabezados.
- Color de fondo: Un azul muy claro y desaturado (#F3F4F6), que proporciona una base limpia y espaciosa que facilita la lectura de los datos.
- Color de acento: Turquesa brillante (#3CC2DD), utilizado para resaltar acciones importantes y gráficos, proporcionando un contraste refrescante.
- Fuente principal: 'Inter' (sans-serif), elegida por su legibilidad y aspecto moderno y objetivo, adecuada tanto para titulares como para el cuerpo de texto, lo que asegura claridad en la presentación de datos.
- Utiliza íconos de contorno simples y reconocibles que representen ubicaciones, búsquedas y detalles de la propiedad, para mantener una interfaz limpia y funcional.
- Diseño limpio y enfocado: Se prioriza un diseño intuitivo con una clara jerarquía visual. La entrada del usuario y la visualización de los resultados están organizadas de manera prominente, evitando distracciones.
- Animaciones sutiles para la carga: Se incluyen animaciones discretas de carga o 'spinners' para indicar que la aplicación está procesando la solicitud, mejorando la experiencia del usuario sin ser intrusivas.
